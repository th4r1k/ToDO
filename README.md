# ToDoApp

A simple JAVA crud with .csv persistance

## Features

- Create;
- Edit;
- Delete;
- Search;
- Organize;
- Order;

### Requisites

Java 8+

### Installing

Clone or download files

Inside the main path run:

    javac -d . Main.java

    java Main
